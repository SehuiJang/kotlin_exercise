# Why Kotlin?
## English
Kotlin is a statically typed JVM-based programming language officially supported by Google for Android development. Since the Android ecosystem has been moving in a Kotlin-first direction, choosing Kotlin for a personal Android app project is a natural decision in terms of official documentation, tooling support, and long-term ecosystem alignment.  

Kotlin also maintains strong interoperability with Java while enabling more concise and safer code. Features such as null safety, data classes, lambdas, extension functions, and type inference help reduce boilerplate and lower the risk of runtime errors in Android app development.  


## Korean
Kotlin은 Android 개발을 위해 Google에서 공식적으로 지원하는 JVM 기반의 정적 타입 프로그래밍 언어입니다. 또한 Android 개발 생태계는 Kotlin-first 방향으로 발전하고 있어, 개인 Android 앱 프로젝트의 기술 스택으로 Kotlin을 선택하는 것은 공식 문서와 도구 지원 측면에서도 자연스러운 선택이라고 판단했습니다.  

Kotlin은 Java와의 상호운용성을 유지하면서도 더 간결하고 안전한 코드를 작성할 수 있게 해줍니다. 특히 null safety, data class, lambda, extension function, type inference 같은 기능은 Android 앱 개발에서 반복 코드를 줄이고 런타임 오류 가능성을 낮추는 데 도움이 됩니다.

## English

That is why I thought Kotlin is suitable for my personal project.
